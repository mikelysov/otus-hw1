Предварительный анализ данных
Домашнее задание № 1
Цель работы. В данном домашнем задании вы потренируетесь в процессе сбора
данных, их предобработки и отображения.

Вам предлагается на основе предоставленной информации:
1. Создать git-репозиторий, где будет храниться исходный код вашего проекта.
Если вы используете приватный репозиторий – дайте преподавателям курса доступ к
нему, для возможности проверки ДЗ.
2. Добавить файл лицензии, который отражает ваш взгляд на конфиденциальность
информации, которую вы подготовите в рамках данного курса.
3. Создать код на Python, который загрузит на ваш локальный компьютер данные о
котировках ценных бумаг из списка SnP500 и котировки криптовалют (BTC, ETH, SOL,
XRP).
4. Поскольку вам предстоит много работать с ними в дальнейшем, подготовьте код,
который будет выполнять автоматическое дополнение данных.
5. Проанализируйте загруженный датасет, если в нем есть пропуски или
несоответствие типов, то подготовьте код, который выполнит предобработку.
6. Проанализируйте выбросы. Оцените, на самом ли деле это выбросы или
реальные данные, с которыми предстоит работать.
Для того, чтобы вашу работу зачли необходимо выполнить 1; 3; 4; 5 пункты
вышеприведенные задания.
